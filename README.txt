Для запуска данного проекта через IntelliJ IDEA необходимо выполнить клонирование проекта, после чего перейти в терминал и прописать команду mvn clean install. 
Если билд проекта прошёл успешно, то далее необходимо пройти в корневую папку проекта, в которой должны находиться файлы DockerFile и docker-compose.yml. 
На вашем ПК также должны быть установлены Docker и Docker Compose. 
После чего необходимо открыть командную строку или PowerShell из корневой директории проекта и прописать команду docker-compose up -d. 
После этого проект должен развернуться в Docker. Если контейнеры запущены, то на данном этапе можно перейти по ссылке http://localhost:8080/swagger-ui/index.html#/ и взглянуть на документацию API. 
Желательно после этого подключиться к базе данных taskman и проверить таблицу users на наличие пользователей. 
Если она оказалась пуста, добавьте несколько пользователей для тестирования API в формате (email, name, password, role) values ('user1@mail.com', 'user1', 'user', 'CLIENT'). 
После этого можно приступить к проверке API через Postman. Для аутентификации необходимо выполнить POST-запрос на http://localhost:8080/authenticate с телом запроса (пример):

json
Copy code
{
  "username": "user1@mail.com",
  "password": "user"
}

После чего, в качестве успешного ответа, вы получите свой JWT-токен, который можно использовать для последующих запросов.
